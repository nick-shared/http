# http
Shared Http helper functions


#### Installation

1. Add this to composer.json:
```
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/nick-shared/php-http.git"
    }
  }
```
2. Add this to composer.json:
```
"require": {
    "mutant/http": "dev-master"
}
```

3. `composer install`
